## Installation

1. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```

2. Run the game:
   ```sh
   python main.py
   ```
3. Unzip resources folder

## Controls

- `WASD` - Move
- `Mouse` - Look around
- `Space` - Shoot

Enjoy the game!
